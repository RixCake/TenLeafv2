# 🌿 TENLeaf Tea — Vietnamese Herbal Wellness Tea

A premium direct-to-consumer ecommerce website MVP for **TENLeaf Tea** — a modern wellness brand importing authentic herbal teas from the highlands of Vietnam.

---

## 🚀 Live Demo

Deploy instantly with **GitHub Pages**:
1. Push this repo to GitHub
2. Go to **Settings → Pages → Source: main branch**
3. Your site will be live at `https://yourusername.github.io/tenleaf-tea`

---

## 📁 Project Structure

```
tenleaf-tea/
├── index.html       # Full website (all pages, styles, and scripts in one file)
└── README.md        # This file
```

All images are embedded as base64 directly in `index.html` — no external assets required. Works offline and deploys to any static host.
